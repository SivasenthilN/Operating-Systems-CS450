1. Save the Code in the Linux environment (Ubuntu) in the name “assignement1.c”
2. Open the terminal and move to the directory where you have saved the file using the cd command in the terminal
3. install the gcc compiler using the following command 
	sudo apt-get install gcc
4. Type the following command to compile the source code
gcc assignment1.c -o work
	in the command mentioned above gcc is the compiler I have used
	assignment1.c is the file name of the source code 
	-o work is the output file for the source code
	This will run the source code without any error
./work 
5. This is execute the source code in which you need to give the input(commands of linux system) to get the outputs
